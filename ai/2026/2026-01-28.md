### Tooling Demos & Updates

#### **AIBOMGen (Elyas & Gopi)**
* **Demo:** Elyas showcased the UI, which generates AIBOMs using Hugging Face repo IDs, ArXiv URLs, or GitHub URLs.
* **Key Features:**
    * **Conflict Detection:** Includes a "conflict" property to flag discrepancies when information from different sources doesn't match.
    * **Graph Visualization:** The tool visualizes the BOM as a graph to assist in composition analysis.
* **Feedback:** Victor suggested coordinating with the [GUAC](https://github.com/guacsec/guac) group to see if they can ingest AIBOMGen data. Gopi noted that graph-based analysis is a primary selling point for the tool.

#### **SysAuditor Tool Suite (Alfred)**
* **Focus:** Supply chain auditing and "Systems Engineering" for regulatory issues.
* **Features:** Data capture, change monitoring, analysis, visualization, and import/export.
* **Discussion:** Alfred and Victor discussed the need to either adapt to downstream tools (like GUAC) or shift the mindset regarding how new SPDX data is consumed.
