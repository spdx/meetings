### Google Search Centre Distribution
- Which profile should address this?
- SPDX may benefit from the same distribution approach as Croissant, using structured data markup that enables automated discovery by search engines and AI systems.
- Google’s structured data framework allows metadata to be embedded in web pages and automatically indexed.
- SPDX documents could potentially be made similarly discoverable when published alongside AI model repositories or system registries.
- This would support the Dynamic Discovery requirements in the SoAS architecture by making governance metadata machine-readable and queryable.
- Reference: Schema Markup Testing Tool | Google Search Central | Google for Developers
- Use the Rich Result Test to see what Google results can be generated for pages and the schema markup validator for generic schema validation.

### Action Item
- Prototype Croissant-compatible SPDX embedding for model repository discovery.
- Volunteer needed.
